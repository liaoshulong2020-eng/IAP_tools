## 🌟 核心内容概览

### 什么是BootLoader？
- 嵌入式设备的固件启动程序
- 支持在线固件升级(IAP)
- 通过Modbus协议与上位机通信

### 主要功能:
1. **系统启动** - 初始化硬件并启动应用
2. **固件升级** - 通过通信接口接收新固件并编程到Flash
3. **验证保护** - 使用CRC32验证固件的完整性
4. **APP跳转** - 验证通过后跳转到应用程序执行

### 支持的通信方式:
- UART (115200 bps) - 串口通信
- CAN (125 Kbps) - CAN总线通信

### 关键技术:
- 实时系统设计
- Flash编程和擦除
- Modbus RTU协议
- CRC校验算法
- 中断处理和定时控制
